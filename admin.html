📣Alertas de Volumen Inusual

Esta función detecta automáticamente cuando el volumen de trading de una criptomoneda supera en N veces su promedio histórico reciente. Un volumen inusualmente alto puede indicar movimientos importantes del mercado, noticias relevantes o actividad institucional. Al configurar el multiplicador, defines cuánto tiene que superar el promedio para que se genere una alerta.

Multiplicador de disparo
Alerta cuando el volumen actual ≥ N × el promedio histórico.
Ej: con 5× — solo alerta si el volumen es 5 veces mayor de lo normal.
Alerta de volumen global
Cuando está activado: el sistema monitorea el volumen de TODAS las monedas en tu lista y genera una notificación cuando alguna supera el multiplicador configurado.
Cuando está desactivado: el sistema ignora el volumen y no genera alertas por este criterio.
Multiplicador activo: 5× — alerta cuando el volumen supere 5 veces el promedio histórico

Proveedor de Datos
FUENTE DE PRECIOS EN TIEMPO REAL
📡Proveedor activo

El proveedor es la fuente de datos de precios que usa Sniper Alert. Cada proveedor tiene diferentes límites de peticiones y cobertura geográfica. Si estás en una región bloqueada por Binance (ej: EE.UU.), cambia a CoinGecko o CoinPaprika que son de acceso global sin restricciones.

Fuente principal
Binance: más rápido, datos directos. Regional (bloqueado en algunos países).
Kraken / CoinGecko / CoinPaprika: acceso global, sin restricciones geográficas.
Intervalo de actualización
Cada cuánto tiempo el sistema pide nuevos precios.
⚠️ Las APIs públicas tienen límites: mínimo recomendado 3s (Binance), 5s (CoinGecko).

💬Canales de aviso

Cuando se cumple una condición de alerta, el sistema puede avisarte por varios canales simultáneamente. Activa o desactiva cada uno según tus preferencias.

Notificaciones del sistema operativo (SO)
Muestra una notificación emergente nativa de Windows/Mac/Linux, incluso cuando la ventana de Sniper Alert está minimizada o en segundo plano. Es el canal más visible.
Banner dentro de la aplicación
Muestra un banner de alerta directamente en la interfaz principal de Sniper Alert con el detalle de la señal (precio de entrada, símbolo, tipo de alerta). Complementa la notificación del SO.
Flash de tarjeta
Hace parpadear en rojo la tarjeta de la criptomoneda que disparó la alerta. Útil para identificar visualmente qué moneda está activa en ese momento. El parpadeo dura 30 segundos.
Duración del flash (segundos)
Cuántos segundos parpadea la tarjeta tras una alerta. Valor mínimo: 5s. Máximo: 300s (5 minutos).
Cooldown & Límites
CONTROL DE FRECUENCIA DE ALERTAS
⏱️Cooldown global

El Cooldown es el tiempo mínimo de espera entre dos alertas consecutivas del mismo tipo para el mismo símbolo. Por ejemplo, con un cooldown de 2 minutos: si BTC dispara una alerta de precio a las 10:00, la próxima alerta de precio para BTC no puede ocurrir antes de las 10:02, aunque el precio siga cumpliendo la condición. Esto evita que el sistema te bombardee con alertas repetidas.

Tiempo mínimo entre disparos (segundos)
Aplica por símbolo y por tipo de alerta (precio, subida %, bajada %, volumen) de forma independiente.
⏱️ 2 minutos entre disparos del mismo tipo
ℹ️ El Supertrend tiene su propio cooldown fijo de 4 horas por señal/símbolo, independiente de este valor.
